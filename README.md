**DS-3001 | University of Virginia**

**Group Members:** Zelalem Asmare, Isaiah Lucero, Andrew Patterson, Simon Alam

## Overview
Analysis of CDC NHANES (1999–2000) health survey data linked with mortality records to predict survival outcomes using k-Nearest Neighbors.

## Goals
- Predict mortality (kNN Classifier)
- Predict life expectancy in months (kNN Regressor)

## Data Sources
- [CDC NHANES 1999–2000](https://wwwn.cdc.gov/nchs/nhanes/continuousnhanes/default.aspx?BeginYear=1999)
- [CDC Linked Mortality File](https://www.cdc.gov/nchs/data-linkage/mortality-public.htm)

## Setup
Download the following files and place them in the same directory as the notebook:
- `linked_mortality_file_1999_2000.csv` — from [course GitHub](https://github.com/ds4e/undergrad_ml_assignments/tree/main/data)
- `DEMO.xpt` — CDC NHANES 1999–2000 Demographic Data
- `BPX.xpt` — CDC NHANES 1999–2000 Blood Pressure Examination Data
- `SMQ.xpt` — CDC NHANES 1999–2000 Smoking & Tobacco Use Data

All NHANES files available at: https://wwwn.cdc.gov/nchs/nhanes/continuousnhanes/default.aspx?BeginYear=1999
